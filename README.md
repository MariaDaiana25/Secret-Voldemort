# ing1_alfabeta


### Run locally

```bash

$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ export AUTHJWT_SECRET_KEY=secret key
$ cd backend/app
$ uvicorn main:app --reload
```
### Execute New Terminal
```bash
$ cd secret-voldemort
$ npm install
$ npm start
```

### Check PEP8

```bash
$ pycodestyle .
$ pytest .
```
